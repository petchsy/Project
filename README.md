# สรุปบทเรียนจากรายวิชา Drone (EE495)
สรุปบทเรียนจากวิชา Drone รหัสวิชา EE495 อาจารผู้สอน  ผศ.ดร.รุจิพรรณ สัมปันณา 
และ รศ.ดร.ณัฐภพ นิ่มปิติวัน วิชานี้จะเรียนรู้เกี่ยวกับการสร้างตัวโดรนขึ้นมาเอง เริ่มจากเข้าใจโครงสร้างของโดรนว่าอะไรทำหน้าที่อะไรบ้าง ไม่ว่าจะเป็นตัวโครง ตัวควบคุม เซนเซอร์ หรือมอเตอร์
